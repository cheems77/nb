== Description ==

<p>RSFiles! is the perfect Wordpress download manager that allows you to:<br/><br/>

    * Share a folder and its items<br/>
    * Add screenshots to files<br/>
    * Add download mirrors to files<br/>
    * Add descriptions, versions and license information<br/>
    * Create groups of users and set up permissions to download and modify files<br/>
    * Use themes to display the downloadable files<br/>
    * Keep track of download counters
</p>


<p><a href="http://www.rsplugins.com/wordpress-plugins/wordpress-download-manager" target="_blank"><strong>RSPlugins.com</strong></a> - RSFiles! - Loads of features that can make your life easier when sharing files to your readers.</p>

== Installation ==

   1. Upload the rsfiles-download-manager folder to the /wp-content/plugins/  directory.
   2. Activate the plugin through the 'Plugins' menu in WordPress.

You will see the RSFiles! menu in the Wordpress Admin Sidebar.
